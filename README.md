# trying-exceptions

What is this program about: 
This program is an economic calendar filter. The user inputs the currency they want to see the news related with the currency of choice. It uses dictionaries as data records, functions and error handling.

What did I learn and struggle with:
1. try/except was used for the very first time with this program. Relatively easy concept to grasp however looking for the place an error could occur was the challenge.

Extra:
